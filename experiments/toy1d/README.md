# Toy 1D Example

This directory contains a notebook that generates figures for the review paper
"Surrogate-Based Bayesian Inference: Uncertainty Quantification and Active Learning"
([arxiv: 2603.13646](https://arxiv.org/abs/2603.13646)). It is kept here for convenience
but is not part of the main paper experiments.

## Contents
- `toy_1d_example.ipynb` — Generates 1D illustrative figures for the review paper
- `utils.py` — Helper utilities used by the notebook
